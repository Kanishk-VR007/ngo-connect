## 🔑 Key Information

### Technology Stack
- **Backend**: Node.js + Express + MongoDB
- **Frontend**: React + React Router
- **Real-time**: Socket.IO
- **Auth**: JWT

### Main Features
✅ User authentication and authorization
✅ NGO discovery and profiles
✅ Service request management
✅ Donation system
✅ Real-time chat
✅ Analytics dashboard
✅ Role-based access control

### User Roles
1. **User** - Browse NGOs, request services, donate
2. **NGO Member** - Manage requests, view analytics
3. **Admin** - Full system access
